# Habit Tracker using Python

A Python application that tracks daily coding hours using the Pixela API.

## Features
- Create a personalized graph
- Record coding hours daily
- Visualize progress over time
- REST API integration

## Technologies Used
- Python
- Requests Library
- Pixela API

## Installation

pip install requests

## Run

python main.py
